Uuu is a multi-functional scientific calculator that is also designed to train you with precalculus/calculus+ and physics skills. It runs by USB-A.
Basically, it's one of these graph calculators, but its way more unique. It has a sound system! You can play any music you like! I made this mainly for it to be a study tool, not a gaming tool. (I might code some games and download some videos though)

(What I'm writing down below is what I'm planning for it to include, this is still a work in progress)
Uuu also offers you quizzes. For example, a quiz about integrals, you can choose the difficulty, and the calculator will generate one for you!
It can also show you detailed explanations about how to do these topics! You can also make your own quizzes, and you can choose if you want to make a game style when playing them!
You can also use lots of various tools! For starters, it has a random number generator, and notepads. It also has a detailed explanation of the periodic table, and a vector field generator!

I made this because I sometimes get procrastinated from studying, so I wanted to make a very fun tool that can play music so you can chill and study at the same time. Again, this calculator isn't just calculating numbers, it can also give you quizzes, teach you, and maybe even play a movie.
